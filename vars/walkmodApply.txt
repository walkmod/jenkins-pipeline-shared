<strong>walkmodApply(body)</strong>

<p>
    Fixes the source code according the patch generated by WalkMod. By default,
    the script pushes the changes it if tests pass.

    The parameter is a closure, that expects values, whose default values are:

    validatePatch = false,
    reportDir = 'target',
    branch = 'master',
    alwaysApply = true
    alwaysFail = true
</p>
